# Engineering Tools Landscape
## IT Systems and Software Tools Inventory

### 1. Requirements Management

**IBM DOORS Next Generation (DNG)**
- Purpose: Requirements capture, management, and traceability
- License: Enterprise floating licenses (50 seats)
- Integration: Linked to Rhapsody for model traceability
- Training: 2-day certification course required

**Jama Connect**
- Purpose: Requirements collaboration with external suppliers
- Usage: Shared projects with Tier-1 suppliers
- Access: Web-based, SSO enabled

### 2. Modeling and Simulation

**IBM Rhapsody**
- Purpose: SysML/UML modeling for MBSE
- Version: 9.0.1
- Profile: Custom automotive profile installed
- Code Generation: C/C++ for embedded targets

**MATLAB/Simulink**
- Purpose: Algorithm development and simulation
- Toolboxes: Control System, Signal Processing, Embedded Coder
- Integration: Model-in-the-Loop testing pipeline

**Enterprise Architect (Sparx)**
- Purpose: Software architecture documentation
- Usage: UML diagrams for software design
- Repository: SQL Server backend

### 3. Project Management

**Microsoft Project Server**
- Purpose: Project scheduling and resource management
- Integration: Synced with SAP for cost tracking

**Jira Software**
- Purpose: Agile sprint management, issue tracking
- Workflows: Custom automotive development workflow
- Plugins: Tempo for time tracking, Zephyr for test management

**Confluence**
- Purpose: Documentation wiki, meeting notes
- Spaces: Per-project spaces with templates

### 4. Configuration Management

**Git (GitLab Enterprise)**
- Purpose: Source code version control
- Branching: GitFlow model enforced
- CI/CD: Integrated pipelines for build/test

**IBM Rational ClearCase**
- Purpose: Legacy code and document versioning
- Status: Migration to Git in progress

**Windchill PDMLink**
- Purpose: Product data management, CAD file control
- Integration: Linked to CATIA and NX

### 5. Testing Tools

**Vector CANoe**
- Purpose: ECU network simulation and testing
- Protocols: CAN, CAN-FD, LIN, Ethernet

**dSPACE ControlDesk**
- Purpose: Hardware-in-the-Loop testing
- Integration: With SCALEXIO real-time platform

**LDRA TBvision**
- Purpose: Static analysis, code coverage
- Standards: MISRA C:2012 compliance checking

### 6. Collaboration

**Microsoft Teams**
- Purpose: Team communication, video conferencing
- Integration: SharePoint document sharing

**Miro**
- Purpose: Virtual whiteboarding, workshops
- Usage: Architecture brainstorming sessions
