# Excel Data Sources

## Required Files:
1. **Qualifizierungsmodule_Qualifizierungspläne_v4_enUS.xlsx** - Main data source containing:
   - Role-Competency Matrix (14×16)
   - Qualification Archetypes (6 strategies)
   - Learning Objectives Templates
   - SE Competence Modules Library
   - Process-Competency Matrix
   - Learning Format Selection Criteria

## File Structure Expected:
- Sheet 1: SE_Competencies_SE4OWL
- Sheet 2: Role-Process-Matrix  
- Sheet 3: Competence-Process-Matrix
- Sheet 4: Role-Competences-Matrix
- Sheet 5: Learning Objectives
- Sheet 6: Module Overview
- Sheet 7: Qualification Archetypes

## Usage:
Place your Excel file here and run: `claude-code run process-excel-data`
