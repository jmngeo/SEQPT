# Setup Complete SE-QPT Project

Initialize the complete SE-QPT project with all dependencies, database, and integrations.

## Command Actions:
1. Verify all input sources are present
2. Create Python virtual environment 
3. Install Flask backend dependencies
4. Setup Vue.js 3 frontend with Vite
5. Initialize PostgreSQL database
6. Process Excel data sources
7. Setup RAG vector database
8. Validate Derik integration compatibility
9. Create development configuration
10. Run initial system tests

## Prerequisites:
- Place Excel file in data/source/excel/
- Set OPENAI_API_KEY in .env
- Ensure PostgreSQL is available

## Expected Output:

- Functional Flask backend on port 5000
- Vue.js frontend on port 3000
- PostgreSQL database with SE-QPT schema
- RAG vector database ready
- All questionnaires loaded and validated
