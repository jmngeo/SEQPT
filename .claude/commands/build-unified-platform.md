# Build Unified Web Platform

Create the complete web platform integrating all SE-QPT components with unified Flask + Vue.js 3 stack.

## Command Actions:
1. Setup Flask backend with all SE-QPT APIs
2. Create Vue.js 3 frontend with Pinia state management
3. Implement 4-phase workflow interface
4. Integrate Derik's assessment components
5. Create RAG-LLM objective generation UI
6. Build admin interface for questionnaire management
7. Setup result visualization with Chart.js
8. Implement PDF export functionality
9. Create user authentication and role management
10. Setup responsive design for mobile compatibility

## Platform Features:
- Phase 1: Maturity assessment and archetype selection
- Phase 2: Competency assessment (Derik's) + RAG objectives
- Phase 3: Module selection and format optimization
- Phase 4: Cohort formation and individual planning
- Admin: Questionnaire editing and system configuration

## Technology Stack:
- Backend: Flask 3.0+ with PostgreSQL
- Frontend: Vue.js 3 + Pinia + Element Plus
- Charts: Chart.js + vue-chartjs
- Export: jsPDF for reports
