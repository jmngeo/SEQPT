# Backup Data

Create comprehensive backups of all SE-QPT data sources and configurations.

## Command Actions:
1. Backup PostgreSQL database with schema and data
2. Export questionnaire configurations to JSON
3. Backup learning objective templates
4. Save RAG vector database state
5. Export Excel processing results
6. Backup user configurations and settings
7. Create configuration snapshots
8. Generate backup verification checksums
9. Test backup restoration process
10. Upload to secure backup storage

## Backup Contents:
- Database: Schema + Data (SQL dump)
- Questionnaires: JSON configuration files
- Templates: Learning objective templates
- Vector DB: RAG knowledge base state
- Configs: All system configurations
- Logs: System operation logs
